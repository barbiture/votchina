Автор: Вадим Зиамбетов

Номинация: Эскизный проект общественной территории поселка Вотчина Country Club

Описание:

В соответствии с замыслом конкурсанта общественная территория поселка будет состоять из двух районов: «Центрального парка» и «Восточного берега».

1. Центральный парк

Предполагается, что поселок должен иметь парк отдыха, который будет использоваться круглогодично. В парк можно прийти пешком без использования автотранспорта с любого конца поселка. Это место, где можно совершать прогулки, пробежки, кататься на велосипеде, роликах, играть с детьми, перекусить в кафе, поплавать на лодке, читать и т.д. Так же есть Wi-Fi. Парк оборудован скамейками, игровой площадкой для детей, туалетом. Территория покрыта газоном и насаждениями в виде цветов, кустарника, деревьев, дорожки покрыты плиткой, имеется два фонтана. Для круглогодичного обслуживания парка имеются подсобные помещения для хранения оборудования и техники, есть магазин, лодочная станция. Центральный парк состоит из двух частей разделяемых рекой, соединяемых мостиками.

Обслуживающий персонал — 3-4 человека.

2. Восточный берег.

Представляет собой пляж. Летом используется для купания, рыбалки, игр, торжественных мероприятий, прогулок, а зимой для прогулок на лыжах, рыбалки, мероприятий, зимнего купания. Имеет место для парковки автотранспорта, помещения для отдыха, переодевания, туалет и душ, а так же подсобных помещений для круглогодичного использования берега, кафе.

Имеется зона для родителей с маленькими детьми, где можно купаться в оборудованной купалке и играть в оборудованном месте, есть навес для укрытия от солнца и помещение. Так же есть оборудованный выход к открытой воде.

Основной пляж имеет оборудованные горки, подходы к воде, переносные лежаки и зонты. Можно играть в различные игры. Имеется площадка со сценой с козырьком и скамейками для торжественных мероприятий и спортивная площадка.

Спортивная площадка состоит из площадки для игры в мини-футбол и волейбольной площадки. Они покрыты специальным резиновым покрытием, а кромка беговой дорожки имеет желобок для улавливания песка с пляжа. Спортивная площадка огорожена сеткой и оборудована скамейками.

Южнее находится домик рыбака и лодочная пристань.

Обслуживающий персонал — 3-4 человека.
